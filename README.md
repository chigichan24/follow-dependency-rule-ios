## For Developers
### Initial setup

1. install Mint

Mint is a package manager for Swift command line tool packages

```
brew tap yonaskolb/Mint https://github.com/yonaskolb/Mint.git
brew install mint
```

2. Run init.sh

```
./init.sh
```

### When Develop

```
mint run XcodeGen generate
```

open FollowingDependencyApp.xcodeproj
